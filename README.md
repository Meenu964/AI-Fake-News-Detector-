# AI-Fake-News-Detector
A Machine Learning-based Fake News Detector for Students that classifies news articles as Real or Fake using NLP and Logistic Regression.
A Full-Stack Web Application built with **Django**, **Machine Learning**, and **Natural Language Processing (NLP)** to classify news articles as Real or Fake, complete with Sentiment Analysis and an Audit History log.

## Features
- Real-time News Authenticity Verification.
- Confidence Score Percentage display.
- Text Sentiment Analysis (Positive, Negative, Neutral) using TextBlob.
- Interactive Audit History Log Dashboard with modern UI formatting.

## Tech Stack
- **Backend:** Django
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **ML & NLP:** Scikit-Learn (Logistic Regression & TF-IDF Vectorizer), TextBlob, Regex
